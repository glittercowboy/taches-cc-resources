<objective>
Reconstruct Reddit discussion threads by linking each comment to its parent post and building a thread context string for each record.

Purpose: Comments about AI in healthcare may not mention those topics directly — they are relevant because they respond to a parent post that does. The LLM relevance classifier (Plan 01-04) must receive parent context alongside the comment text. This plan builds that context field.
Output: `data/processed/02_with_thread_context.rds` — all records augmented with `thread_id`, `parent_text`, and `full_context` fields.
</objective>

<task type="auto">
  <name>Task 1: Create R/thread_reconstruct.R — parent linking and thread ID assignment</name>
  <files>R/thread_reconstruct.R</files>
  <action>
    Create `R/thread_reconstruct.R` with a function `reconstruct_threads(df)`.

    **Background on Reddit `parent_id` format:**
    - `parent_id` for a top-level comment (direct reply to post) is `t3_{post_id}` where `t3_` is Reddit's prefix for posts (link objects)
    - `parent_id` for a nested reply is `t1_{comment_id}` where `t1_` is the prefix for comments
    - Root posts have `parent_id == NA` or empty

    **Implementation:**
    1. Strip Reddit type prefixes: `clean_parent_id = gsub("^t[0-9]+_", "", parent_id)`
    2. For each record, derive `thread_id`:
       - For posts (`type == "post"`): `thread_id = id`
       - For top-level comments: `thread_id = clean_parent_id` (this is the post ID)
       - For nested comments: walk up the parent chain to find the root post ID. Use an iterative approach with a lookup table (id → clean_parent_id) to avoid deep recursion. Limit chain walk to 10 levels to avoid infinite loops on malformed data.
    3. Build a lookup of `id → title + text` for posts and `id → text` for comments (use `coalesce(title_and_body, text)`)
    4. For each record, add:
       - `parent_text` (character): direct parent's text (NA for root posts)
       - `root_post_text` (character): the root post's combined title + body (same as text for posts)
       - `thread_depth` (integer): 0 for posts, 1 for direct replies, 2+ for nested
    5. Log: number of records with resolved parent, number with unresolved parent (parent not in dataset — expected when parent post was filtered out)
    6. Return augmented tibble

    **Edge cases to handle:**
    - `parent_id` is NA → root post, skip parent lookup
    - Parent ID not found in dataset (filtered out earlier) → set `parent_text = NA`, `root_post_text = NA`, `thread_depth = NA_integer_`. These will still go through relevance classification with their own text only.
    - Circular references (malformed data) → detect with cycle check, assign `thread_depth = NA_integer_`
  </action>
  <verify>`source("R/thread_reconstruct.R"); df_threaded <- reconstruct_threads(df_filtered); "parent_text" %in% names(df_threaded); "thread_id" %in% names(df_threaded)`</verify>
  <done>All records have thread_id, parent_text, root_post_text, and thread_depth fields; posts have thread_depth == 0; direct replies have thread_depth == 1; log reports resolution rate</done>
</task>

<task type="auto">
  <name>Task 2: Build full_context field and run thread reconstruction pipeline script</name>
  <files>R/thread_reconstruct.R (updated), scripts/02_thread_reconstruct.R</files>
  <action>
    Add a function `build_full_context(df)` to `R/thread_reconstruct.R` that creates a `full_context` character field for each record — this is what the LLM will receive for relevance classification.

    **Context composition rules:**
    - For root posts (`thread_depth == 0`):
      `full_context = paste0("[POST in r/", subreddit, "]\nTitle: ", title, "\n\n", text)`
    - For direct comments (`thread_depth == 1`, parent is a post):
      `full_context = paste0("[COMMENT in r/", subreddit, " replying to post]\nParent post title: ", parent_title, "\nParent post body: ", truncate(root_post_text, 500), "\n\n[COMMENT TEXT]\n", text)`
    - For nested comments (`thread_depth >= 2`):
      `full_context = paste0("[NESTED COMMENT in r/", subreddit, "]\nRoot post title: ", root_title, "\nDirect parent: ", truncate(parent_text, 300), "\n\n[COMMENT TEXT]\n", text)`
    - For unresolved (`thread_depth == NA`):
      `full_context = paste0("[COMMENT in r/", subreddit, " — parent context unavailable]\n", text)`

    Helper `truncate(text, max_chars)`: trims to max_chars with "..." appended if truncated.

    Create `scripts/02_thread_reconstruct.R`:
    ```r
    source("R/utils.R"); load_config()
    source("R/thread_reconstruct.R")

    df_filtered <- readRDS(file.path(CFG$path_processed, "01_initial_filtered.rds"))
    df_threaded <- reconstruct_threads(df_filtered)
    df_threaded <- build_full_context(df_threaded)

    saveRDS(df_threaded, file.path(CFG$path_processed, "02_with_thread_context.rds"))
    log_msg(paste("Thread reconstruction complete:", nrow(df_threaded), "records;",
                  sum(!is.na(df_threaded$parent_text)), "with resolved parent context"))
    ```
