Use the GPTZero API to detect and flag likely AI-generated Reddit discussion texts, then remove flagged records from the pipeline dataset.

Output: `data/processed/04_human_filtered.rds` — relevant records with AI-generated content removed; `data/processed/04_gptzero_audit.rds` — full results including flagged records.
</objective>

  <name>Task 1: Create R/gptzero_filter.R — API integration with quota tracking and checkpointing</name>
  <files>R/gptzero_filter.R</files>
  <action>
    Create `R/gptzero_filter.R` with the following components:

    **GPTZero API details:**
    - Endpoint: `POST https://api.gptzero.me/v2/predict/files` (bulk) or `POST https://api.gptzero.me/v2/predict/text` (single)
    - Auth header: `x-api-key: {GPTZERO_API_KEY}`
    - Request body for bulk: `{"documents": [{"id": "...", "text": "..."}]}` (up to 10 documents per call)
    - Response: per-document `completely_generated_prob` (0–1), `average_generated_prob`, `predicted_class` ("human"/"mixed"/"ai")
    - Use `httr2` for HTTP calls

    **1. `count_words(text)`** — simple word count: `length(strsplit(trimws(text), "\\s+")[[1]])`.

    **2. `estimate_word_quota(df)`** — sums word counts across all records; logs estimated words to be sent; warns if > 80% of monthly quota.

    **3. `call_gptzero_bulk(documents_list, api_key)`** — sends up to 10 documents to the bulk endpoint via `httr2`, returns parsed response. Handles HTTP errors with `httr2::resp_check_status()`. Returns list of per-document results.

    **4. `run_gptzero_filter(df, cfg = CFG, ai_threshold = 0.75)`** — main function:
    - Check for existing checkpoint with `load_latest_checkpoint("gptzero")`; skip already-processed IDs
    - Pre-flight: call `estimate_word_quota(df)` and log; warn if quota risk
    - Process in batches of 10 records (bulk endpoint limit)
    - For each batch: call `call_gptzero_bulk()`, extract `completely_generated_prob` and `predicted_class` per document
    - Save checkpoint every 200 records
    - Track running word count; if approaching monthly limit (> 280k words), stop gracefully and save checkpoint with a warning message
    - Return tibble: `id`, `gptzero_prob` (numeric 0–1), `gptzero_class` (character), `is_ai_generated` (logical: `gptzero_prob >= ai_threshold`)

    **Threshold note:** 0.75 is the default. Add a comment: "Lowering to 0.5 captures mixed-AI content; raising to 0.9 only removes highly confident AI. Adjust CFG if needed."

    Add `gptzero_ai_threshold = 0.75` to `config/config_template.R`.

<task type="auto">
  <name>Task 2: Create scripts/04_gptzero_filter.R and apply AI-detection filter</name>
  <files>scripts/04_gptzero_filter.R</files>
  <action>
    Create `scripts/04_gptzero_filter.R`:
    ```r
    source("R/utils.R"); load_config()
    source("R/gptzero_filter.R")

    df_relevant <- readRDS(file.path(CFG$path_processed, "03_relevance_classified.rds"))

    # Pre-flight quota check
    total_words <- sum(sapply(df_relevant$text, count_words))
    log_msg(paste("Total words to analyse:", format(total_words, big.mark=","),
                  "/ Monthly limit: 300,000"))
    if (total_words > 300000) {
      log_msg("WARNING: Dataset exceeds monthly quota. Consider processing in batches across months,
               or filtering to highest-priority records first.", level = "WARN")
    }

    # Run GPTZero
    gptzero_results <- run_gptzero_filter(df_relevant)

    # Join results
    df_gptzero <- df_relevant |>
      dplyr::left_join(gptzero_results, by = "id")

    # Summary
    n_ai       <- sum(df_gptzero$is_ai_generated == TRUE, na.rm = TRUE)
    n_human    <- sum(df_gptzero$is_ai_generated == FALSE, na.rm = TRUE)
    n_no_score <- sum(is.na(df_gptzero$is_ai_generated))
    log_msg(paste("GPTZero results — AI-flagged:", n_ai,
                  "| Human:", n_human, "| Unscored:", n_no_score))

    # Save audit file (all records with scores)
    saveRDS(df_gptzero, file.path(CFG$path_processed, "04_gptzero_audit.rds"))

    # Save human-only for downstream pipeline
    df_human <- dplyr::filter(df_gptzero, is_ai_generated == FALSE | is.na(is_ai_generated))
    saveRDS(df_human, file.path(CFG$path_processed, "04_human_filtered.rds"))
    log_msg(paste("Final human-filtered dataset:", nrow(df_human), "records"))
    ```

    Note: records with `is.na(is_ai_generated)` (unscored due to quota exhaustion) are kept by default — conservative approach. Add a comment explaining this choice.

    Also add a `DRY_RUN` flag (first 20 records only) at top for testing without burning quota.
