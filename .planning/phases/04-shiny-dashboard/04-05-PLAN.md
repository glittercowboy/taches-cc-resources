    **Record detail panel** â€” shows full text and all labels when a row is clicked:
    ```r
    output$record_detail <- shiny::renderUI({
      sel <- input$table_records_rows_selected
      if (is.null(sel) || length(sel) == 0) {
        return(shiny::p("Click a row to see full record details.", class = "text-muted"))
      }
      # Get the selected record's ID from the DT proxy
      df_preview <- filtered_records() |> dplyr::distinct(id, .keep_all = TRUE)
      rec_id <- df_preview$id[sel]
      rec <- dplyr::filter(filtered_records(), id == rec_id)

      # Build detail UI
      shiny::tagList(
        shiny::tags$p(shiny::tags$strong("Subreddit:"), paste0("r/", rec$subreddit[1])),
        shiny::tags$p(shiny::tags$strong("Date:"), format(rec$created_utc[1], "%Y-%m-%d")),
        shiny::tags$p(shiny::tags$strong("Type:"), rec$type[1]),
        shiny::tags$p(shiny::tags$strong("Keyword:"), rec$keyword[1]),
        shiny::hr(),
        shiny::tags$p(shiny::tags$strong("Text:")),
        shiny::tags$blockquote(rec$text[1], style="font-size:0.9em; border-left:3px solid #ccc; padding-left:10px;"),
        shiny::hr(),
        shiny::tags$p(shiny::tags$strong("Transformer Sentiment:"),
                      shiny::tags$span(rec$text_sentiment[1],
                        class = paste0("badge bg-", switch(rec$text_sentiment[1],
                          positive="success", negative="danger", neutral="secondary", "warning"))
                      )),
        shiny::tags$p(shiny::tags$strong("Emotion:"), rec$emotion_label[1]),
        shiny::hr(),
        shiny::tags$p(shiny::tags$strong("ABSA Aspects:")),
        # Show all aspects for this record as badges
        do.call(shiny::tagList, lapply(seq_len(nrow(rec)), function(i) {
          if (is.na(rec$aspect_label[i])) return(NULL)
          shiny::div(class = "mb-1",
            shiny::tags$span(rec$aspect_label[i], class = "badge bg-info me-1"),
            shiny::tags$span(rec$aspect_sentiment[i],
              class = paste0("badge bg-", switch(rec$aspect_sentiment[i],
                positive="success", negative="danger", neutral="secondary", "warning")), class = "me-2"),
            shiny::tags$small(rec$evidence[i], class = "text-muted")
          )
        }))
      )
    })
    ```

    ## Prerequisites
    - R >= 4.1 with renv installed

    - GPTZero API key (set as GPTZERO_API_KEY env var in .Renviron)

    ## Setup
    1. Clone repo, open in RStudio
    2. `renv::restore()` â€” installs all R packages
    3. Copy `config/config_template.R` â†’ `config/config.R` and fill in values
    4. Place raw CSV files in `data/raw/`

    ## Pipeline Execution Order

    | Step | Script | Description | Output |
    |------|--------|-------------|--------|
    | 1 | `scripts/01_initial_filter.R` | Date, dedup, length filter | `01_initial_filtered.rds` |
    | 2 | `scripts/02_thread_reconstruct.R` | Link comments to parent posts | `02_with_thread_context.rds` |
    | 3 | `scripts/03_classify_relevance.R` | LLM relevance classification | `03_relevance_classified.rds` |
    | 4 | `scripts/04_gptzero_filter.R` | AI-generated text detection | `04_human_filtered.rds` |
    | 5 | `scripts/05_run_absa.R` | ABSA labelling via Claude | `05_absa_raw.rds` |
    | 6 | `scripts/06_normalise_absa.R` | Aspect normalisation | `06_absa_labelled.rds` |
    | 6b | `scripts/export_for_inference.R` | Export CSV for Python | `for_inference.csv` |
    | 7 (Python) | `scripts/run_inference.py` | Transformer sentiment/emotion | `07_transformer_scores.parquet` |
    | 8 | `scripts/07_integrate_results.R` | Merge all results | `08_full_pipeline_output.rds` |
    | 9 | `scripts/08_prepare_dashboard.R` | Pre-compute dashboard aggregations | `09_dashboard_data.rds` |

    Steps 3, 5 (and the Python Step 7) all support resuming from checkpoint. Simply re-run the script â€” already-processed records are skipped.
